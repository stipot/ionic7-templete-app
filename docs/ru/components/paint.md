# Компонент Paint
Этот компонент представляет собой полнофункциональный графический редактор, позволяющий пользователям рисовать, создавать фигуры, заливать области и сохранять свои творения.

## Основные функции
- Инструменты рисования: кисть, ластик, пипетка для выбора цвета с холста
- Геометрические фигуры: прямоугольник, круг, треугольник с возможностью заливки
- Работа с цветом: палитра предустановленных цветов + выбор любого цвета через color picker
- Регулировка размера: удобный ползунок для изменения толщины кисти и ластика (1-30px)
- Масштабирование: лупа с увеличением от 50% до 300% и возможность перемещения по холсту
- Отмена действий: кнопка отмены и горячая клавиша Ctrl+Z (хранится до 20 последних действий)
- Управление холстом: очистка всего рисунка с подтверждением
- Сохранение: экспорт рисунка в PNG с автоматическим именованием по дате и времени
- Автосохранение: рисунок автоматически сохраняется каждые 10 секунд и восстанавливается при перезагрузке страницы
- Поддержка touch: рисование пальцем на мобильных устройствах

## Техническая реализация
Компонент использует:
- Angular для реактивной логики и управления состоянием
- Ionic Framework для UI компонентов (ion-header, ion-content, ion-button, ion-range, ion-grid)
- TypeScript для строгой типизации и надежности кода
- Canvas API для отрисовки и манипуляции с пикселями
- SCSS для адаптивных стилей и анимаций
- LocalStorage для автосохранения рисунков
