# Учебное пособие по компоненту «Погода» (Weather Component)

> Этот компонент поможет разработчикам легко реализовать виджет погоды в своих Ionic/Angular-приложениях. Компонент предоставляет актуальную информацию о погоде, прогноз на неделю и поддерживает мультиязычность.

Этот компонент отображает данные из следующих переменных и API-сервисов:
- `weatherData` - переменная, содержащая текущие данные о погоде (температура, влажность, скорость ветра, иконка)
- `weeklyForecast` — массив, содержащий прогноз погоды на 7 дней
- `currentCity` — текущий выбранный город (по умолчанию "Moscow")
- `WeatherAPI` — внешний API-сервис (weatherapi.com) для получения данных о погоде
- `TranslateService` — сервис для интернационализации (поддержка русского, английского, французского, итальянского языков)

Фильтр «Город». Этот фильтр поможет пользователю выбрать любой город для отображения погоды. Реализован через AlertController и ModalController.
Фильтр «Температура» может помочь отобразить текущую температуру в градусах Цельсия (°C).
Фильтр «Ощущается как» рассчитывает и показывает комфортную температуру с учётом влажности и скорости ветра.
Фильтр «Прогноз на неделю» отображает данные на 7 дней: дневная температура, ночная температура, дата (день недели + число), иконка погодных условий.
Фильтр «Автообновление» автоматически обновляет данные о погоде каждые 10 минут (600 000 мс).
