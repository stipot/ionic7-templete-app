# Разработка компонента "VideoEditor"

Компонент управляет состоянием видеоплеера, обрабатывает выбор файлов, реализует логику обрезки таймлайна и покадровую перерисовку видео на canvas для сохранения результата с фильтрами.
Реализован перевод.
Реализовано изменение формата видео.
onTrimChange(): Обновляет границы обрезки и перематывает видео для предпросмотра.
onTimeUpdate(): Следит за тем, чтобы воспроизведение не выходило за границы обрезанного фрагмента.
applyTrim(): Фиксирует выбранный временной диапазон.
