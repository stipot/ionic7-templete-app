# Разработка компонента "Магазин-одежды"

>Этот компонент разрабатывался как магазин с выбором типа одежды и добавлением его в корзину для покупок.

## Для реализации данного компонента использовались такие методы Ionic как:
<ion-segment> - используется для фильтрации товаров по категориям. Позволяет пользователю быстро переключаться между разделами (Футболки, Джинсы, Куртки и т.д.)
<ion-card> - используется для отображения каждого товара. Содержит:
    Цветную плитку с названием товара (вместо изображения)
    Название и категорию товара
    Цену в рублях
    Выбор размера через <ion-select>
    Кнопку добавления в корзину
<ion-grid> и <ion-col size="6"> - используется для создания двухрядной сетки товаров. Адаптивно отображает по 2 товара в ряд на мобильных устройствах.
<ion-badge> - отображает количество товаров в корзине на иконке корзины
<ion-item-sliding> - используется в корзине для возможности свайпа и удаления товара
localStorage - используется для сохранения корзины между сессиями
TranslateService - используется для мультиязычности (русский, английский, французский, итальянский)
[style.backgroundColor] - динамическое изменение цвета плиток для каждого товара
