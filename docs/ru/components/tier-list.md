# Учебное пособие по компоненту «Тир-лист»

> Этот компонент поможет пользователям оценивать и сортировать любые элементы ввиде картинок (персонажей, фильмы, игры) по уровням, от лучшего к худшему, визуализируется cледующим образом - S,A,B,C,D,Е. Компонент поддерживает перетаскивание картинок, загрузку собственных изображений и сохранение результата в png формате.

Этот компонент отображает данные из следующих переменных:
- sourceImages — массив, содержащий исходные картинки для левой панели (предустановленные и загруженные пользователем изображения)
- placedItems — массив в котором находятся картинки, распределенные по рядам

Основные элементы интерфейса:

- Левая панель с картинками
В этой панели находятся все доступные для перетаскивания картинки. Пользователь может захватить любую картинку мышью и перенести её в один из рядов.
- Ряды S, A, B, C, D, E
Каждый ряд соответствует своему уровню качества:
S — лучшие
A — очень хорошие
B — хорошие
C — средние
D — плохие
E — худшие
- Пользователь может перетаскивать картинки из левой панели в любой ряд, а также перемещать уже размещённые картинки между рядами
- Крестик удаления
На каждой картинке, размещённой в ряду, есть крестик, при нажатии который картинка удаляется из ряда
- Кнопка <Загрузить картинку>
Позволяет пользователю загрузить свою собственную картинку с устройства
- Кнопка <Сохранить>
Таблицу с рядами(делает скриншот) сохраняет как png-файл
