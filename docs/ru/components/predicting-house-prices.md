# Модель машинного обучения для предсказания цен

## Описание
В данной программе используется модель линейной регрессии. По сути самая простая и примитивная модель. Она была обучена в отдельном python файле на датасете под названием [Moscow Housing Price Dataset](https://www.kaggle.com/datasets/egorkainov/moscow-housing-price-dataset)

## Ошибки
В программе есть несколько ошибок, которые выводятся пользователю:
- Ошибка: общая площадь не может быть меньше чем: (жилая площадь + площадь кухни)!
- Ошибка: количество этажей в доме не может быть меньше, чем текущий этаж!
- Не удалось посчитать, попробуйте другие параметры  
(Последняя ошибка связанна с тем, что модель может посчитать отрицательную цену или 0)

## Что можно улучшить?
- попробовать избавиться от ошибок
- попробовать очистить данные в датасете, что бы улучшить результат линейной регрессии
- использовать другой датасет
- использовать более сложную модель
