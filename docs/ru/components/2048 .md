#   Компонент "2048 Game"
Этот компонент представляет собой классическую головоломку 2048, где игрок объединяет плитки с одинаковыми числами для достижения числа 2048.
# Основные функции
Игровая механика: перемещение плиток стрелками клавиатуры, объединение одинаковых чисел
Смена стилей: 4 темы оформления (классическая, тёмная, цветная, минимализм)
Разные размеры поля: 3×3, 4×4, 5×5 с соответствующими целевыми значениями (512, 2048, 4096)
Система очков: подсчёт текущего и лучшего результата

# Техническая реализация
Компонент использует:
Angular для реактивной логики игры
Ionic компоненты для интерфейса (ion-header, ion-button, ion-content)
TypeScript интерфейсы для типизации данных плиток
SCSS для адаптивных стилей и анимаций
