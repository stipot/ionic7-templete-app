# Учебное пособие по компоненту «Спидометр скорости интернета»

Компонент позволяет измерять реальную скорость интернета (Download и Ping) с помощью API Cloudflare. Результат отображается на анимированном спидометре с цветовой индикацией.

---

## Компонент отображает данные из следующих переменных:

- downloadSpeed - текущая скорость скачивания в Мбит/с
- ping - время отклика сервера в миллисекундах
- isTesting - флаг, показывающий, идёт ли тестирование

---

## Основные функции

### 1. Тест скорости Download

Скачивает тестовый файл 5 МБ с сервера Cloudflare, замеряет время и вычисляет скорость.

Алгоритм:
- Выполняется 3 замера
- Отбрасываются минимальное и максимальное значения
- Итоговая скорость = среднее арифметическое

### 2. Тест Ping

Отправляет HEAD-запрос к favicon.ico текущего приложения и замеряет время ответа.

### 3. Визуализация на спидометре

- Стрелка поворачивается от -90° (0 Мбит/с) до +90° (300 Мбит/с)
- Шкала размечена от 0 до 300 Мбит/с с шагом 50

---

## Цветовая индикация

### Скорость Download

- 0 Мбит/с - серый
- менее 30 Мбит/с - красный
- 30-60 Мбит/с - оранжевый
- более 60 Мбит/с - зелёный

### Пинг

- 0 мс - серый
- менее 30 мс - зелёный
- 30-90 мс - оранжевый
- более 90 мс - красный

### Текстовый статус

- 0 Мбит/с - Не измерено
- менее 10 Мбит/с - Очень медленно
- 10-30 Мбит/с - Нормально
- 30-60 Мбит/с - Хорошо
- более 60 Мбит/с - Отлично

---

## Используемые API

- https://speed.cloudflare.com/__down?bytes=5000000 - тест Download
- window.location.origin + '/favicon.ico' - тест Ping
