# Документация компонента «Флорист» (Plant Care Manager)

> Этот компонент представляет собой интерактивную панель управления комнатными растениями, реализованную на фреймворке Ionic с использованием Angular. Он предназначен для визуального мониторинга коллекции растений и контроля графика их полива. Компонент использует адаптивную сетку Ionic (Grid) для отображения карточек растений и модальные окна для управления данными.

## Ключевые особенности

*   **Трекинг полива в реальном времени:** Автоматический расчет дней до следующего полива. При истечении срока плитка растения визуально выделяется (класс
`needs-water`), а индикатор меняется на предупреждающий.
*   **Управление коллекцией:** Возможность быстрого добавления новых экземпляров через модальное окно и удаление существующих.
*   **Интерактивные действия:** Кнопка быстрого полива обновляет статус растения без необходимости заходить в детальное меню.
*   **Адаптивная верстка:** Сетка автоматически перестраивается под разные размеры экранов (от мобильных телефонов до планшетов) благодаря параметрам `size-md` и `size-lg`.
*   **Обработка отсутствующих данных:** 
    *   Визуальный экран «пустого состояния» (Empty State), если список растений пуст.
    *   Автоматическая подстановка заглушки (Placeholder), если ссылка на изображение растения невалидна или отсутствует.

## Структура интерфейса

1.  **Header:** Содержит навигацию, заголовок с поддержкой локализации и кнопку вызова модального окна добавления.
2.  **Plant Grid:** Сетка карточек, каждая из которых содержит:
    *   Индикатор статуса (Badge) с иконкой капли или предупреждения.
    *   Фотографию растения с обработкой ошибок загрузки.
    *   Текстовую информацию (Название, Вид).
    *   Панель действий (Полить/Удалить).
3.  **Add Plant Modal:** Форма с полями ввода для настройки параметров нового растения (имя, вид, периодичность, фото).

## Технические требования

Для корректной работы компонента требуются следующие зависимости:
- **Ionic Framework** (использование компонентов `ion-card`, `ion-grid`, `ion-modal`).
- **ngx-translate:** Для обработки пайпов `| translate`.
- **FormsModule:** Для двусторонней связки данных (`ngModel`) в форме добавления.
