# Gazon Component File Converter

## is a simple and convenient tool for converting files between different formats. It supports working with images (JPG, PNG, WebP, GIF, etc.) and documents (PDF, DOC, TXT, etc.).

## Main Features:
1) Download files via drag-and-drop or select from the system;
2) Automatic detection of the source file format;
3) Conversion between popular formats;
4) Downloading the finished file;

## Interface Features:
1) Intuitive 3-step process;
2) Visual cues and icons;
3) Responsive design for all devices;
4) Instant feedback on the result;
5) The component is integrated into the main menu of the application and is ready for use.
