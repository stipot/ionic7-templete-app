# 2048 Game Component

This component is a classic 2048 puzzle where the player merges tiles with the same numbers to reach 2048.

# Main Features

- **Game mechanics**: Moving tiles using keyboard arrows, merging identical numbers
- **Style switching**: 4 design themes (classic, dark, colorful, minimalism)
- **Different field sizes**: 3×3, 4×4, 5×5 with corresponding target values (512, 2048, 4096)
- **Scoring system**: Current and best score tracking

# Technical Implementation

The component uses:
- **Angular** for reactive game logic
- **Ionic** components for interface (ion-header, ion-button, ion-content)
- **TypeScript interfaces** for tile data typing
- **SCSS** for responsive styles and animations

# Data Architecture

```typescript
interface Tile {
    value: number
    id: number
    row: number
    col: number
    isNew?: boolean
    merged?: boolean
}
```
