# Конвертер файлов (Gazon Component)

## Простой и удобный инструмент для конвертации файлов между различными форматами. Поддерживает работу с изображениями (JPG, PNG, WebP, GIF и др.) и документами (PDF, DOC, TXT и др.).

## Основные возможности:
1) Загрузка файлов через drag-and-drop или выбор из системы;
2) Автоматическое определение формата исходного файла;
3) Конвертация между популярными форматами;
4) Скачивание готового файла;

## Особенности интерфейса:
1) Интуитивно понятный процесс из 3 шагов;
2) Визуальные подсказки и иконки;
3) Адаптивный дизайн для всех устройств;
4) Мгновенная обратная связь о результате;
5) Компонент интегрирован в основное меню приложения и готов к использованию.
