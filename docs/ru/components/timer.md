 # Разработка компонента "таймер"


 > этот компонент разрабатывался для настройки обратного отсчета, визуализации оставшегося времени и сохранения истории недавних таймеров.


Для реализации данного компонента использовались такие методы Ionic, как:
- <ion-grid>, <ion-row> и <ion-col>: используются для создания адаптивной сетки, позволяющей расположить поля ввода часов, минут и секунд в один ряд.
- <ion-item>: выступает в качестве контейнера для элементов ввода, обеспечивая правильные отступы и визуальное разделение в списке.
- <ion-input>: используется для ввода числовых значений и названия таймера. 
- <ion-button>: используется для управления состоянием таймера (Старт, Стоп, Сброс)
- <ion-list>: используется для отображения секции «Недавние таймеры» с возможностью быстрого повторного запуска.

Что надо доделать:
Кнопка быстрой очистки: добавить одну кнопку «Очистить всё», которая обнуляет название и все поля времени одним нажатием.
Сортировка истории: настроить список недавних таймеров так, чтобы последние запущенные всегда отображались.
